principais Marcaçoes

Negrito

podemos deixar um texto ou uma pequena parte em negrito apenas usando a tecla B

texto em itálico

Podemos tambem colocar nossos textos em italicos usando a tag I.

Marcaçoes

E caso haja necessidade, podemos colocar nossos textos em marcaçoes usando a tecla 'MARK'

E se quisermos, ainda conseguimos trocar a cor do marca texto para uma de nossa preferencia. Usando a tag de CSS, 'Style'

Principais Formataçoes

Codigo-fonte

Para colocarmos codigo no nosso texto e fazer-los com oque todos ocupem o mesmo espaço pode ser usar a tag 'code'

Exemplo: num1 = int(input("Digite outro número: ")) num2 = int(input("Digite outro número: ")) soma = num1 + num2 print("A soma de {} e {}, é igual a:".format(num1,num2,soma))

Como podemos ver, cada letra do paragrafo acima possuem o mesmo tamanho.

Porem, o codigo nao esta com a indentaçao correta, entao para colocarmos ele com a indenteçao correta, devemos colocar a tag 'PRE'.

Exemplo: 
             
            num1 = int(input("Digite outro número: "))
            num2 = int(input("Digite outro número: "))
            soma = num1 + num2
            print("A soma de {} e {}, é igual a:".format(num1,num2,soma))
